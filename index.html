📊 Conectar con Google Sheets

Seguí estos pasos UNA sola vez. Después la sincronización es automática.

1
Creá un Google Sheet nuevo en sheets.google.com. Nombralo "Visitas Emmapel".
2
En el Sheet, andá a Extensiones → Apps Script.
3
Borrá todo lo que hay y pegá este código exacto:
function doPost(e){ try{ var s=SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); if(s.getLastRow()===0){ s.appendRow(['Timestamp','Fecha','Turno','Vendedor', 'Cliente','Rubro','Resultado','Monto ($)', 'Motivo no compra','Notas','ID visita']); } var d=JSON.parse(e.postData.contents); s.appendRow([ new Date().toLocaleString('es-AR'), d.fecha||'',d.turno||'',d.vendedor||'', d.cliente||'',d.rubro||'',d.resultado||'', d.monto||'',d.motivo||'',d.notas||'',d.id||'' ]); return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); }catch(err){ return ContentService .createTextOutput(JSON.stringify({ok:false,error:err.toString()})) .setMimeType(ContentService.MimeType.JSON); } } function doGet(e){ return ContentService .createTextOutput(JSON.stringify({ok:true,msg:'Emmapel activo'})) .setMimeType(ContentService.MimeType.JSON); }
4
Guardá el proyecto (Ctrl+S). Luego hacé clic en Implementar → Nueva implementación.
5
En "Tipo", elegí Aplicación web. En "Quién tiene acceso" elegí Cualquier usuario. Hacé clic en Implementar. Autorizá el acceso con tu cuenta Google.
6
Copiá la URL de la aplicación web que aparece al final y pegala acá abajo:
⚠️ La URL empieza con https://script.google.com/macros/s/... — si no empieza así, revisá el paso 5.
